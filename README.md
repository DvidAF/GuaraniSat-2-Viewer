## GuaraníSat-2 3D Viewer

Interactive web application built with **Three.js** to visualize the **GuaraníSat-2 nanosatellite** in 3D.
It allows users to explore the model with dynamic controls, realistic lighting, and optimized performance for web environments.

---

## Features

- Interactive 3D visualization of a GLB model
- Auto-rotation with toggle control
- Camera controls using OrbitControls
- Soft shadows and improved lighting setup
- Optimized space background using JPEG
- Draco-compressed 3D model for faster loading
- Automatic camera framing based on model size
- Modern UI with loading screen and controls

---

## Technologies Used

- [Three.js](https://threejs.org/)
- GLTFLoader + DRACOLoader
- WebGL
- JavaScript (ES Modules)
- HTML5 + CSS3
